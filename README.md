# PanVK Driver Plugin
A Vulkan Driver Plugin For Zalith Launcher That Enables The Use Of Vulkan API On Unsupported Devices With Mali GPUs. PanVK Is Basically Turnip But For Mali GPUs. It Makes Vulkan Work On Unsupported Devices By Replacing The System Vulkan Driver With Mesa's PanVK Driver That Can Fill The Extension Or Feature Gaps In System Vulkan Driver.

# Usage

PanVK Driver Plugin Is Mainly Made For Mali Devices That Don't Support The Required Extensions And Features For Running Minecraft 26.2+ And Snapshots Which Use Vulkan API.
